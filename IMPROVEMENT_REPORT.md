## 2. 電話占いアフィリエイト戦略

### 主要ASP

| ASP | 案件数 | 特徴 |
|---|---|---|
| **A8.net** | 業界No.1 | 高単価案件最多 |
| **afb** | 6案件以上 | ヴェルニ、ピュアリ、絆など |
| **アクセストレード** | 中程度 | 単価高め |
| **Link-A** | 一部高単価 | Feelが特に高い |

### 報酬単価相場

| 成約条件 | 報酬 | 承認率 |
|---|---|---|
| 無料会員登録のみ | 3,000〜5,000円 | **60〜90%（高い）** |
| 無料登録+初回利用 | 5,000〜15,000円 | 10〜30%（低い） |

→ **「無料会員登録のみ」の案件を選ぶのがコスパ最強**

### 推奨サービスTOP3（実装済み）

1. **電話占いヴェルニ** - 創業20年・実力派多数
2. **電話占いウィル** - 芸能人も利用
3. **電話占いカリス** - 占い師合格率3%

### 効果的な導線設計（実装済み）

1. **占い結果表示直後** ← 最も効果的
   - 感情が動いている瞬間
   - 「もっと知りたい」欲求が最大

2. **カテゴリ別メッセージ**
   - 夢占い → 「夢の本当の意味、もっと詳しく」
   - 相性占い → 「お相手との未来、もっと詳しく」
   - 前世占い → 「前世の物語、もっと詳しく」

### 法的対応（実装済み）

- **PR表記をファーストビューに表示**（ステマ規制対応）
- **footer に サイト共通のPR表記**
- **`rel="sponsored"` 属性**

### 月10万PVでの収益試算

```
10万PV × CTR 1% = 1,000クリック
1,000クリック × CVR 5% = 50成約
50成約 × 平均報酬 4,000円 × 承認率 70% = 140,000円/月
```

電話占いアフィリエイトだけで月14万円の試算。
